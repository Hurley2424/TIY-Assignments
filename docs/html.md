Block-level HTML tags, their attributes, and some example usage.

1. `<ol>` -- ordered list

example:
<ol>
  <li>one thing</li>
  <li>two things</li>
  <li>three things</li>
</ol>

2. `<ul>` -- unordered list

example:

<ul>
  <li>one thing</li>
  <li>two things</li>
  <li>three things</li>
</ul>

3. `<address>` -- contact information

Example:

<address>
    You can contact me at <a jeremy.bouchard@gmail.com </a>. <br>
</address>
